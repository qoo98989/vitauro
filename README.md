# 輝達泌尿科診所 — 線上排班系統 (VITA Uro Clinic Scheduler)

雲端即時同步的月班表系統。所有人開同一個網址即可檢視,店長可編輯,員工可線上預假,改動即時同步到每個人的畫面。

## 🔗 使用網址
https://qoo98989.github.io/vitauro/Vita_v13.html

打開即可使用,免登入、免安裝。手機、平板、電腦皆可。

## 🧭 三種模式
| 模式 | 說明 | 進入方式 |
|------|------|----------|
| 檢視 | 唯讀,看班表(預設) | 打開網址即是 |
| 管理 | 完整編輯班表、設定 | 點「🔒 管理」輸入密碼 |
| 預假 | 員工點格切換「休」 | 管理模式按「開放預假」,全裝置同步進入 |

## ✨ 主要功能
- 28 天月班表,點格排班、拖曳一次選多格
- 自動加總總時數 / 月休,對照目標
- 班別:早 / 早1 / 晚 / 晚2 / ALL / 休(週六店休、週日無早1)
- 個別員工可上班別、時數、備註設定
- **雲端即時同步**(Firebase Realtime Database):一人改、所有人自動更新
- **全域預假開關**:店長一鍵開放/結束,所有員工手機同步
- 匯出 PNG 圖片、匯出 / 匯入 JSON 備份
- 深色模式、手機版自適應、觸控操作

## ☁️ 資料與同步
- 班表資料儲存於 Firebase Realtime Database(asia-southeast1 新加坡)
- 左上角同步燈:🟢 已同步 / 🟠 離線(沒網路時顯示上次內容,恢復後同步)
- **第一個連上的裝置會把其班表上傳為雲端基準**,首次請由店長在正確班表的裝置開啟

## ⚠️ 重要:定期備份
資料庫為開放讀寫,任何有網址者皆可編輯。請**定期按「匯出備份」存 JSON 檔**;若資料被誤改,進管理模式按「匯入備份」即可還原。

## 🔒 隱私
- 本檔內含的 Firebase apiKey 為公開金鑰,屬正常設計,不構成機密。
- 班表資料不儲存於 GitHub,僅存於 Firebase。
